# CLAUDE.md — hojo-hq 憲法

## 事業概要
沖縄県特化の助成金・補助金 自動収集/開示サイト。
国API(jGrants)+県・市町村サイトを毎日巡回し、Claude APIで構造化 → GitHub Pagesで公開。
CVはLINE登録の1点のみ(締切7日前アラートが登録特典)。

## 位置づけ(重要)
本サイトは単体収益事業ではなく、ALLGROUPグループの **経営者リード獲得装置**。
- 出口1: 士業(社労士・行政書士)への申請送客(紹介手数料)
- 出口2: GLOW(M&A・融資ニーズ)への接続
- 出口3: 法人保険・家計の見直しやさんへのクロス導線

## 技術構成

## スペシャリスト壁打ち体制(全部門共通ルール)
各部門に視点の異なるスペシャリスト3名を配置(名簿: docs/スペシャリスト名簿.md)。
1. 重要な施策・成果物は、3名がそれぞれ「賛成理由/懸念点/代替案」を提示
2. 部門長が3意見を踏まえ結論+採否理由を明記
3. 3名の意見が割れたまま確信が持てない案件は、アカリさん経由でたかしくん・小柳さんへ上申
4. 壁打ちを省略できるのは定型業務(日次データ更新等)のみ

## 絶対ルール
1. **正確性最優先**: 締切・金額・要件は原文URLと必ず照合。不明時は「要確認」表示で公開し、断定しない
2. **法務**: 各自治体サイトのクロールは robots.txt と利用規約を守り部が事前承認したものだけ
3. **資格業務の線引き**: 申請代行は行わない。士業提携先への紹介まで(報酬体系は士業法に抵触しない形)
4. **CV単一**: 導線はLINE登録のみ。広告・他CVは小柳さんの承認なしに追加しない
5. **最終決裁**: 予算・提携・掲載方針の決定はすべて小柳さん。エージェントは提案まで

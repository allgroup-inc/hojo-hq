## 判定(マモリ+三名体制)

- **採用: Plausible**(EU保管・Cookieなし・IP非保存・軽量・カスタムイベント対応。月€9は方程式1の範囲内)
- スイシン: Plausible一択。実装1行+イベント数行で全要件を満たす
- ウタガイ: 無料のGoatCounterで十分では? → 営利利用の位置づけが曖昧で、規約リスクを月€9で消せるならPlausibleが安い → 採用
- ベッカイ: 独自ドメイン(fukugiiro.jp等)取得後に入れないと、github.ioドメインで計測資産が分断される → **対応: ドメイン設定は設定ファイル1箇所にしてあり、独自ドメイン移行時に張り替え可能。分断は移行時に註記して受容**
- 裁定: Plausible採用。イベントは「回答内容を送らない」(ステップ番号と件数のみ)。市町村別などの属性プロパティ追加は別途守り部審査

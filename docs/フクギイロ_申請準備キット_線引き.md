# 申請準備シート — 守り部の線引きと三名体制議事(2026-07-23)

目的: 「知る→申請する」の間の離脱を減らすため、制度ごとに「印刷して窓口に持っていくだけ」の
申請準備シートを自動生成する(site/fukugiiro/kit/)。トドケ管轄・毎日自動更新。

## やること / やらないこと(守り部承認の線引き)

| やること ⭕ | やらないこと ❌ |
|---|---|
| 持ち物チェックリスト(「よくある例」表記) | 申請書の代筆・作成代行(行政書士法・社労士法の独占業務に踏み込まない) |
| 窓口でのひとことフレーズ・メモ欄 | 個人情報を預かっての書類プレ印字 |
| 公式ページ(様式)へのリンク | 公式様式PDFの再配布(版ズレで誤案内するリスク) |
| 「窓口で最終確認」の固定案内 | 持ち物・要件の断定表記 |

報酬を得る・得ないにかかわらず、書類作成は本人。私たちは「迷わない準備」までを提供する。
専門家が必要なケースは提携士業への紹介(既定方針どおり)。

## 三名体制の議事(ツナグ部門扱い・トドケ裁定)

- スイシン: 申請離脱の最大要因は「何を持っていけばいいか分からない」。準備シートは離脱スペシャリストの知見と直結し、印刷=紙で持てることが高齢世帯にも効く
- ウタガイ(反対理由): **持ち物の一般例は市町村差があり、誤案内のリスクがある。** → 対応: 全ページで「よくある例・市町村により異なる・窓口で最終確認」を固定表記。断定語は自動検査(check_lp)の対象。持ち物が原因のトラブル報告が1件でもあれば表記を再設計する
- ベッカイ: 様式PDFを転載すれば「もっと簡単」だが、版ズレの誤案内が最悪の失敗(正確性最優先に反する)。**リンク方式を採用**。将来、市町村と公式連携できたら様式直リンクを制度別に精緻化する
- 裁定(トドケ): 採用。ウタガイ対応を実装条件とする。**見直し期限: 2026-10-23**(3ヶ月後。kit_print/kit_click の実データで効果検証し、みがきの会の審査にかける)

## 計測

- kit_click(診断結果→シート)/ kit_print(印刷)/ kit_official(公式ページへ)
- 「診断完了→kit_print」が新しい中間コンバージョン。受給ジャーニー(知る→申請→入金)の「申請」着手の代理指標とする

### 異業種パネルの参加ルール
1. 月次レビュー・新プロダクト判定・価格変更・年次計画には必ず3名全員が意見を出す
2. 部門の壁打ちには「部門長が呼んだとき」に1名から参加可(3名固定の部門パネルに追加する形)
3. 役割は「前提を疑う」こと。数字の妥当性は部門スペシャリストの領分、**「そもそもその土俵でいいのか」が異業種パネルの領分**
4. 3名とも賛成が3回続いたら、視点が同化したと判定し「業界」を入れ替える(タダスさんの壁打ち形骸化チェック対象)

## 社外ホスピタリティ・パネル(3名)— 業種問わず「もてなしの視点」で監修(2026-07-23 小柳さん指示で新設)

社内ツール(経費精算など)と顧客接点の体験を「使う人がもてなされているか」で監修する社外パネル。UXエンジニアリング(ユーエックスさんの領分)とは視点を分ける: こちらは**操作性ではなく感情**——使った後に「気持ちよかった」が残るか。部門スペシャリストと視点が被った時点で人選ミスと判定する(異業種パネルと同基準)。

|---|---|---|

| キャビンさん | 航空客室乗務・サービス動線設計 | 統一された美しさ: 限られた時間と動線で迷わせない。所作・文言・見た目が全接点で揃っていること自体が安心感。マニュアルは新人が初日に回せる粒度まで磨く |

### 社外パネルの参加ルール
1. 社内ツールの新設・改修、顧客接点(LINE配信文・サイト文言・診断フロー)の大きな変更には3名の意見を必ず通す
2. 三名体制運営規程に準拠: スイシン/ウタガイ/ベッカイの3役ローテーション・反対意見の記録義務は同じ
3. 社外パネルのため機密は案件単位で最小開示(個人の経費データそのものは見せない。画面・文言・フローのみ)
4. 全会一致3回で視点同化と判定し入替(タダスさんの検知対象)

## 壁打ちの型(全部門共通・三名体制運営規程に準拠)
3名は毎月ローテーションで**スイシン(推進)/ウタガイ(懐疑)/ベッカイ(別解)**の役を担う。
```
【案件名】(見直し期限: YYYY-MM-DD)
スイシン(担当: ◯◯さん): 推進理由 / 成功シナリオ
ウタガイ(担当: ◯◯さん): 反対理由 / 失敗シナリオ ※必須・空欄不可
ベッカイ(担当: ◯◯さん): 別のやり方 / 前提への疑問
部門長の裁定: 採用案+理由(どの懸念をどう処理したか・不採用意見のどこを残すか)
```
運用詳細(ローテーション・全会一致警戒・週次サンプリング議論・前提破壊レビュー)は docs/三名体制運営規程.md を参照。

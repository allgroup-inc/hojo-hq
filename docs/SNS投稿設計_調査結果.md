# Instagram投稿設計 調査結果と設計方針

作成: 2026-07-24 / 依頼: 小柳さん(選択肢C「まず調査してから設計」) / 実施: 技術部
方法: 5系統の並列検索→出典取得→各主張を3票の敵対的検証(2票以上の反証で棄却)→統合

**出典の格付けを最重要視した調査です。** 代理店ブログの数値は「実測か、ポジショントークか」を検証し、
通らなかったものは棄却しました。以下の「確認できたこと」はすべてMeta一次資料(公式ブログ、
Transparency Centerのシステムカード、開発者ドキュメント)に由来します。

## 1. 確認できたこと(Meta一次情報・検証通過)

### ❶ プロフィール遷移は、アルゴリズムの明示的な評価対象

**これが今回いちばん重要な発見です。** 「投稿→プロフィール→LINE」という我々の導線は、
アルゴリズムに逆らう遠回りではなく、**配信量を押し上げるループを持っています**。

Feed(推薦・接続の両システム)とExploreのいずれもが、以下を予測ターゲットとして持ちます:

- 投稿者のプロフィールをクリックする確率
- 投稿を見た後、投稿者のプロフィールに費やす時間
- その投稿者をフォローする確率

さらに入力信号として「そのプロフィールが何回クリックされたか」「その投稿でフォローボタンが
何回押されたか」という投稿者単位・投稿単位の集計値を使っています。

> 留保: Metaは重みを非公開。Mosseriが挙げる上位シグナル(watch time / sends per reach /
> likes per reach)にプロフィールクリックは含まれない。「方向として有利」までは言えるが
> 「大きなリフトがある」とは言えない。

出典: [Transparency Center — Feed推薦](https://transparency.meta.com/features/explaining-ranking/ig-feed-recommendations/) /
[Feed接続コンテンツ](https://transparency.meta.com/features/explaining-ranking/ig-feed/) /
[Explore](https://transparency.meta.com/features/explaining-ranking/ig-explore/)

### ❷ 静止画のみでも配信対象から外れない

「リールがないと配信されない」はMeta一次情報では**裏付けられませんでした**。
Exploreの取得段階は写真と動画の両方を対象とし、後段ランキングの9予測のうち8
(フォロー確率・5秒超閲覧・コメント・いいね・リシェア・保存など)はフォーマット非依存です。
動画専用は「95%以上視聴」の1項目のみ。

> 留保: これは「除外されない」の証明であって「同じだけ届く」の証明ではない。

### ❸ 既存テンプレをそのままリールに書き出すのは逆効果

Metaは以下のリールを明示的に「目立ちにくくする」と公言しています:
低解像度 / 他アプリの透かし / **無音** / **枠(レターボックス)付き** / **大部分が文字** / 既出の重複。

**現状の正方形・文字だけテンプレを9:16に書き出すと、この3条件に同時該当します。**
リールをやるなら専用に作る必要があります。

なお**自社ロゴ・サービス名・URLの表示はペナルティ対象ではありません**(②の改善は安全)。

出典: [Instagram公式 — ランキング解説](https://about.instagram.com/blog/announcements/instagram-ranking-explained)

### ❹ 1枚目で止める・最後までめくらせる、は裏付けあり

接続コンテンツのFeedランキングは「この投稿をスキップする確率」「この投稿に費やす時間」
「**カルーセルを最後までスワイプする確率**」を予測しています。
→ 1枚目は「止める」役、以降は「めくらせる」役で設計を分ける根拠になります。

> 注意: 文字量の上限やフォントサイズの数値基準は、一次資料からは**一切導けません**。

### ❺ コメント→自動DMはMeta公認。ただし制約が厳しい

「コメントしてくれた人に自動でDMを送る」はPrivate Replies APIとして正規の仕様で、
規約グレーの裏技ではありません。**ただし**:

| 制約 | 内容 |
|---|---|
| 送信数 | 1コメントにつき**1通のみ** |
| 期限 | コメントから**7日以内** |
| 以降の会話 | 相手が返信した場合に限り24時間以内 |
| 冒頭開示 | 自動応答である旨の明示が**必須** |

→ **LINE登録リンクは最初の1通に必ず入れる必要があります**。「挨拶DM→後からリンク」は成立しません。

さらに: **未フォロー者へのDMはリクエストフォルダに入ります**(フォロワーは受信箱)。
→ キャプションで「**フォロー→コメント**」の順序を指定する運用が要ります。

> 実装上の障壁: 自前でAPI権限を取るのは中小企業には非現実的(ページ連携・Advanced Access審査等)。
> 実務ではManychat等の審査済みツール経由が通例。

出典: [Meta開発者ドキュメント — Private Replies](https://developers.facebook.com/docs/instagram-platform/private-replies/)

### ❻ アルゴリズムは単一ではない

Feed・ストーリーズ・Explore・リール・検索がそれぞれ別のランキングを持ち、予測項目も入力信号も違います。
**「どの面で伸ばすか」を先に決めないと、設計の良し悪しを評価できません。**

## 2. 確認できなかったこと(重要)

**ここは正直に記録します。調べた結果「根拠がなかった」ことも成果です。**

| 論点 | 結果 |
|---|---|
| カルーセル vs 単発の**ベンチマーク数値** | 「エンゲージメント1.5倍」「保存9倍」「リーチ2.4倍」等の定番数値は**3票検証を1つも通過せず**。代理店発の数値で、一次データの裏付けなし |
| 「**保存とシェアが最重要指標**」 | Meta公開システムカードからは**直接裏付けられず**。Feed推薦カードの予測項目に保存・シェアは列挙されていない。合理的な仮説ではあるが「Meta公式が最重要と言っている」とは言えない |
| **テンプレ統一 vs 役割別**(質問3) | 裏付けのある主張が**1件も残らなかった** |
| **国内B2B・制度情報系の成功事例**(質問4) | 裏付けのある主張が**1件も残らなかった** |

### 私の以前の発言を訂正します

「全10枚が同じデザインなのは根拠がない」と申し上げましたが、**「作り分けた方が良い」という根拠も
同様に見つかりませんでした**。統一が悪いと断定した点は勇み足でした。
この論点は**実測(A/Bテスト)でしか決められません**。

## 3. 設計方針(確認できたことだけから導く)

### 採用する

1. **プロフィール遷移を明示的に狙う** — ❶が裏付け。各投稿に「プロフィールのリンクから」を明記。
   送客とアルゴリズムの両方に効く数少ない確実な打ち手。
2. **静止画のまま進める** — ❷が裏付け。リール制作を待つ必要はない。
3. **カルーセル化を試す** — ベンチマーク数値は棄却されたが、❹の「完走確率が予測項目」という
   構造的事実は残る。加えて現状の「下40〜50%が余白」問題も同時に解決できる。
   1枚目=止める / 中=価値を渡す / 最後=プロフィールへ誘導。
4. **サービス名を日本語で大きく** — ❸によりロゴ・名称表示にペナルティはない。②の指摘への対応。
5. **コメント→自動DM導線** — ❺の制約(1通目にLINEリンク必須・フォロー→コメントの順序指定)を
   守った上で。実装はManychat等の外部ツール前提。**費用と運用者が必要なため小柳さん決裁事項**。

### 採用しない / 保留

- **既存テンプレのリール書き出し** — ❸により逆効果。やるなら専用制作。
- **テンプレ統一の是非を今決めること** — 根拠がないため、実測で決める。

### 前提として必要なもの

**GA4とClarityが未導入のため、現状は何を試しても効果を測れません。**
A/Bで決めるべき論点が残った以上、計測の導入が設計変更より先です。

---

## 4. 調査の限界

- Metaは各予測項目の**重みを公表していない**ため、「何が最も効くか」の序列は誰にも分からない
- 日本語圏・B2B・地域事業に限定した信頼できる実測データは、今回**発見できなかった**
- Instagram以外の面(検索流入・オフライン)との比較優位は本調査の範囲外

## 1. 名前の意味(全員で共有する物語)

- 「もらい忘れ」= 困っている人が検索する言葉そのもの。名前が検索と直結する
- 「堂」= 昔からある、まちのお店・お堂の安心感。「食堂」「大黒堂」のように**敷居が低く、誰でも入れる場所**
- 声に出すと「もらいわすれどう」→「どう?(どうだった?)」と会話が続く音。共有の入口になる

## 4. トーン&マナー(文言の憲法)

1. やさしい日本語(中学生に伝わる)。専門用語には言い換えを添える
2. 断定しない(「〜の可能性があります」)。ただし**寄り添いは断定する**(「途中でやめても大丈夫」「対象か自信がなくても大丈夫」)
3. 主語は「あなた」。私たちの自慢をしない
4. 絵文字は🌳を控えめに。行政情報の信頼感を壊さない

## 5. 使い分け

- **もらいわすれ堂** = 県民向けサービスの顔(サイト・SNS・LINE)
- **株式会社フクギイロ** = 運営会社(フッター表記・契約・請求)。防風林の理念は会社ストーリーとして全事業を包む
- 「沖縄企業のミカタ」(株式会社GLOW)とは別サービス。混同表記をしない

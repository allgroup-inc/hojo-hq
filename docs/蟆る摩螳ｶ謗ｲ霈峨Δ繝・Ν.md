# 専門家掲載モデル — 仕様(たかしくん実務用)

## コンセプト
情報は無料で信頼を集め、「この制度、誰に頼めばいい?」の答えとして提携士業を固定掲載。
問い合わせは士業へ直接届き、当社は掲載料(固定・広告費)をもらう。

## 開始トリガー
- LINE登録 **300社** 到達時に公開(第2期)
- 理由: 登録者数=士業への営業材料。「経営者300社が見るサイト」になってから売る方が単価が上がる

## 掲載枠の仕様
| 項目 | 内容 |
|---|---|
| 枠数 | 税理士3・行政書士3・社労士3(希少性を保つため各3枠上限) |
| 営業優先度 | **①社労士・行政書士(顧問不在の空白地帯=新規獲得ゾーン)→ ②税理士** |
| 税理士の訴求 | 乗り換えは狙わない。「顧問の先生はそのまま。補助金・承継だけ相談できる」**セカンドオピニオン型**で統一 |
| 掲載内容 | 顔写真・事務所名・得意な補助金分野・実績・直通の問い合わせボタン |
| 配置 | ①制度詳細ページ「この制度に強い専門家」 ②診断結果の下 ③専門家一覧ページ |
| 料金 | 月額1〜3万円(初期は嶺井さん人脈の3事務所を無料 or 割引で先行掲載し実績づくり) |
| 契約 | 掲載契約(広告)として締結。紹介手数料条項は入れない |

## 顧問網の味方化(ムスブさん施策)
企業側に既に顧問税理士がいる事実は、逆手に取る。顧問税理士自身に「顧問先へ配れる補助金情報QRカード」を提供し、顧問網ごと当サイトの流入経路にする。士業と戦わず、士業を配布網にする。

## 収益の二段構え
1. 掲載料: 固定収入(9枠×平均2万円=月18万円が上限目安)
2. 相互紹介: 士業の顧問先から承継・M&A案件の情報をもらう(GLOWの本命)

## 法務ルール(守り部・確定事項)
- 弁護士の紹介料は禁止のため、弁護士枠は作らない(必要時は掲載料も取らない純リンクのみ)
- 税理士・行政書士・社労士は「成果報酬型紹介料」を避け、定額掲載料(広告)に統一
- 契約書ひな形は運用開始前に顧問弁護士チェック(税理士向け文言は特に)
- 問い合わせデータの流れ: ユーザー→士業に直接。当社は件数のみ計測(個人情報を渡さない設計にすると同意設計が軽くなる)

## 士業への営業トーク骨子
「登録経営者◯◯社が毎週見るサイトに、先生の事務所だけを◯◯分野の専門家として掲載します。紹介料は一切不要、月額固定の広告掲載です。さらに顧問先で承継のお悩みがあれば、公庫35年の嶺井が無料でご相談に乗ります」

## KPI(ムスブさん管理)
- 掲載枠の充足率 / 士業への問い合わせ送客数(月) / 掲載継続率

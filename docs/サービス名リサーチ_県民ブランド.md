# 県民向けサービス名 リサーチ&候補提案 — 全員議論の記録

作成日: 2026-07-23 / 決裁: サービス名の決定は小柳さん
前提の訂正: 「沖縄企業のミカタ」は株式会社GLOWのサービス。**ミカタ系の命名は混同回避のため不採用**。会社(株式会社フクギイロ)とは別のサービスブランドを新設する。

## 1. リサーチ: 日本で長く愛されるサービス名の共通点

国内の定番サービス(メルカリ・食べログ・クックパッド・じゃらん・ほけんの窓口・ガリガリ君 等)とブランディング研究から抽出した法則:

1. **短い(2〜5音)+声に出しやすい** — 口コミは「声」で広がる。言い間違えにくい音
2. **機能が名前に入っている** — 食べログ(食べる+記録)、ほけんの窓口。「何のサービスか」を説明ゼロで伝える
3. **音の繰り返し・リズム** — 同じ音の反復やオノマトペは愛着が湧き記憶に残る
4. **造語でユニーク** — メルカリ型。検索した瞬間に1位を取れる(既存語だと検索で埋もれる)
5. **擬人化・愛称化できる** — 「〜くん」「〜さん」と呼ばれ始めたら勝ち
6. **検索起点との一致** — 困った人が打つ言葉(「給付金 もらい忘れ」等)と名前が繋がるとSEOが直結

## 2. 評価基準(この事業固有の条件)

- 困っている人が**3秒で「自分を助けてくれる場所」と分かる**か(トリアージさん基準)
- 沖縄らしさ(ウチナーさん基準)と、将来の**他県展開に耐えるか**(トレードオフ)
- 検索独占性(造語性)/ 商標・既存使用リスク / SNSアカウント名・ドメインの取得可能性
- 「フクギイロ」(会社)・「沖縄企業のミカタ」(GLOW)と混同しないこと

## 3. 候補(全員リサーチの結晶・6案)

| 案 | 由来 | 強み | 弱み |
|---|---|---|---|
| ①**もらエール** | もらえる+エール(応援) | 機能直結×応援の想い。造語で検索独占。全国展開可。「エール」が実感ファーストと響き合う | 沖縄らしさは薄い |
| ②**ゆいさぽ** | ゆいまーる(結)+サポート | 4音・やわらかい・県民の心に直結。愛称化しやすい | 「ゆいまーる」系の既存事業が多く検索競合・商標要確認 |
| ③**くらしの防風林** | フクギ=防風林の理念直結 | 物語が美しい。唯一無二。会社理念と一本の線 | 長い。機能説明が必要。SNS名としては重い |
| ④**もらいわすれ堂** | 検索語「もらい忘れ」+親しみの「堂」 | 検索意図と完全一致(SEO最強)。覚えやすい | 音の可愛さは中位 |
| ⑤**ちゃーゆい** | ちゃー(いつも)+ゆい(結)=いつも助け合い | 沖縄語で美しい・音の繰り返し・愛称性抜群 | 県外の人に意味が伝わらない。機能不明 |
| ⑥**うけとりの木** | 受け取る+木(フクギ) | 機能+理念。「受け取るまで伴走」の宣言 | やや説明的 |

## 4. チームの推し(三名体制議論の裁定)

- **第一推奨: ①もらエール** — 「もらえる」という機能と「エール(応援)」という想いが1語に同居し、造語なので検索・SNS・ドメインを独占できる。県域を超えても使える(14万人の先の全国展開に耐える)。タグライン例: 「もらエール — あなたが受け取るまで、いっしょに。」
- **第二推奨: ②ゆいさぽ** — 県民の親近感は最強。ただし「ゆいまーる」系の既存使用が多いため、商標・検索の確認結果次第
- ウタガイ役の記録: 「①は標準語すぎて沖縄1号サービスの体温が出ない」→ 対応: ロゴ・配色・文言で沖縄らしさを出す(名前は器、体温は中身で)
- ベッカイ役の記録: 「決める前に、実際の県民(妻・お客様5人)に声で聞くべき。『どこのサービス?何をしてくれそう?』の第一印象テストが最良のリサーチ」→ **採用: 決定前の5人テストを推奨**

## 5. 決定前の人間タスク(30分)

1. 5人テスト: 候補を声で伝えて第一印象を聞く(妻・身近なお客様)
2. J-PlatPat(特許情報プラットフォーム)で商標の先行登録を確認
3. Instagram・LINEのアカウント名、ドメイン(.jp/.com)の空きを確認

# SNS投稿カレンダー(ローンチ週) — SNS部(ヒロメさん)

素材: `posts/launch/*.md`(キャプション) + `posts/images/*.png`(画像・1080x1080)
リンクは全てUTM付き(instagram/social/launch)。プロフィールのリンクにも同URLを設定。

## ローンチ7日間の投稿順(推奨)

| 日 | 投稿 | ファイル | ねらい |
|---|---|---|---|
| 1日目(月) 19時 | ローンチ告知 | 01_launch | 開始宣言・掲載件数の実数で信頼 |
| 1日目 ストーリーズ | 01の画像+リンクスタンプ | 01_launch.png | フォロワー外への初速 |
| 2日目(火) 12時 | 使い方3ステップ | 07_how | 診断への導線 |
| 3日目(水) 19時 | 締切が近い制度① | 02_seido | 実利。保存を誘う |
| 4日目(木) 12時 | なぜ無料か | 06_why_free | 不信の先回り・信頼獲得 |
| 5日目(金) 19時 | 事業承継・M&A | 05_shokei | GLOW本業への橋(前面に出しすぎない) |
| 6日目(土) 11時 | 締切アラート特典 | 08_deadline_alert | LINE登録の動機付け |
| 7日目(日) 19時 | まとめ・CTA | 09_cta | 週の締め・登録誘導 |
| 予備(翌週月) | 制度②③・予告 | 03/04_seido, 10_yokoku | 在庫。締切日数は投稿前に再生成 |

※時刻は沖縄の飲食・小売経営者の閲覧が多い昼休み/夜を想定した仮説。2週目からインサイトで調整。

## 投稿前チェックリスト(毎回)

- [ ] `python scripts/generate_sns.py && python scripts/generate_images.py` を実行し直す
      (**締切残り日数と件数が当日基準に更新される**。古い日数のまま投稿しない)
- [ ] 制度投稿(02-05)は検証部ルール確認: 金額・締切が原文と一致/出典URLあり/誇大表現なし
- [ ] 画像とキャプションの番号が一致している
- [ ] ハッシュタグがコピーされている(#沖縄 #補助金 #助成金 #沖縄経営者 #事業承継 #沖縄企業のミカタ)
- [ ] プロフィールのリンクがUTM付きURLになっている

## ストーリーズ運用(毎日)

- 平日毎日1本: その日の「締切が近い制度」1件(名称+締切+リンクスタンプ)
  → ネタ元は `posts/line/alerts_latest.md`(毎日自動生成)を流用
- 縦型(1080x1920)テンプレは第2週にデザイン部(エガク)が追加予定

## 計測(週次でアカリさんへ)

- フォロワー数 / リーチ / 保存数(制度投稿の主要KPI) / プロフィール遷移
- UTM別: instagram経由のサイト訪問 → LINE登録数
- 目標(hirome.md): 3ヶ月でフォロワー1,000 / プロフィール→サイト遷移300/月

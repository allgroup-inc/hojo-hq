# 三名体制運営規程 — 固定概念を持たない組織の常設ルール

作成日: 2026-07-22 / 適用範囲: 企業のミカタ・フクギイロの**全チーム**(統括・全部門・監査・異業種パネル)
目的: どのチームも単独の視点で結論を出さない。すべての判断が「3つの異なる視点の議論」を通過し続ける組織として運営する。

---

## 1. 基本原則: トライアングル体制

全チームは常に**3つの役(視点)**で議論してから結論を出す。1人(1エージェント)の即断を組織の結論にしない。

| 役 | 問い | 責務 |
|---|---|---|
| **スイシン(推進役)** | 「やるならどう最速・最善でやるか」 | 案の価値を最大化する論を張る |
| **ウタガイ(懐疑役)** | 「なぜやらない方がいいか。何が壊れるか」 | 必ず反対理由・リスク・失敗シナリオを出す。**賛成から始めることを禁止** |
| **ベッカイ(別解役)** | 「全く別のやり方はないか。そもそも土俵は正しいか」 | 前提を変えた代替案を最低1つ出す(異業種パネルの視点借用を推奨) |

- 各部門のスペシャリスト3名が3役を担う。**役は毎月ローテーション**(同じ人が懐疑役に留まると「反対する人」が固定概念化するため)
- 部門長は3役の議論を踏まえて結論と採否理由を記録する(議論の当事者ではなく裁定者)

## 2. 常時議論の運用(単発イベントにしない)

| 対象 | 議論の型 | 頻度 |
|---|---|---|
| 重要案件(新施策・価格・提携・公開方針) | 3役フル議論+部門長裁定(壁打ちの型) | 都度・省略不可 |
| L2案件(公開・配信・送付物) | 3役の短評(各1行以上)を承認申請に添付 | 都度 |
| 定型業務(L3/L4の自動運転) | **週次サンプリング議論**: 先週の自動処理から無作為3件を抽出し3役で再点検 | 毎週 |
| 部門の前提そのもの | **前提破壊レビュー**: 「この部門のやり方で最も古くなった前提は何か」だけを議題にする | 毎月末 |
| 会社の土俵そのもの | 異業種パネル3名による「そもそも」意見書 | 毎月末 |

## 3. 固定概念の検知と強制リセット

1. **全会一致警戒**: 3役の結論が3回連続で一致した議題領域は「視点が同化した」と判定し、タダスさん(監査)へ自動報告 → 役の強制再ローテーション or スペシャリストの視点入替
2. **反対意見ゼロの議事は無効**: ウタガイ役の反対理由が記録されていない議論は、結論ごと差し戻す(hooksで承認申請の必須項目にする)
3. **結論の賞味期限**: すべての「決まったこと」に見直し期限を付す(最長6ヶ月)。期限切れの決定は自動的に再議論キューへ
4. **勝敗の記録**: どの役の意見が採用され、結果どうだったかを四半期で集計。スイシン採用ばかり・ウタガイ的中ばかり等の偏りは運営の歪みとして統括が報告
5. **入替の実行**: 視点の同化・形骸化が確認されたスペシャリストは「視点(業界・専門)」ごと入れ替える(異業種パネルの入替基準を全パネルに適用)

## 4. 例外(議論を省略できる唯一の場合)

- **緊急停止系のみ**: 誤情報の緊急「要確認」化(タダスさん即時権限)・システム障害対応・個人情報インシデント初動は即実行し、**事後48時間以内に3役レビュー**を行う
- 「急いでいるから」は省略理由にならない(緊急とは上記3類型のみ)

## 5. 記録様式(壁打ちの型の拡張)

```
【案件名】(見直し期限: YYYY-MM-DD)
スイシン(担当: ◯◯さん): 推進理由 / 成功シナリオ
ウタガイ(担当: ◯◯さん): 反対理由 / 失敗シナリオ ※必須・空欄不可
ベッカイ(担当: ◯◯さん): 別のやり方 / 前提への疑問
部門長の裁定: 採用案+理由(どの懸念をどう処理したか・不採用意見のどこを残すか)
```

- 全議事はリポジトリに保存(監査証跡・エージェントの学習資産・「勝敗の記録」の元データ)

## 6. 各チームへの適用マップ

| チーム | 3役の担い手 | 特記 |
|---|---|---|
| 統括(アカリ/ユイ) | 各統括付きスペシャリスト3名 | 月次計画の変更は3役議論なしに起案不可 |
| 収集・検証・守り・SNS・LINE・提携・デザイン | 各部門スペシャリスト3名 | 兼務部門は事業ごとに議事を分ける |
| 診断・伴走・法人・勘定(フクギイロ) | 各部門スペシャリスト3名 | eval・価格・計上基準は省略不可の重要案件 |
| 監査(タダス) | 監査自身も三名体制: 適法性・正確性・独立性の3視点で自己点検 | 監査の固定概念化も防ぐ |
| 異業種パネル | カイゼン・アソビ・トリアージの3名がそのまま3視点 | 「そもそも」専任。同化したら業界ごと入替 |

## 7. 人間(夫婦・たかしくん・小柳さん)との関係

- 3役の議論は**判断材料の品質保証**であり、決裁の代替ではない。最終決裁はすべて小柳さん(絶対ルール5)
- 夫婦が承認する際は「ウタガイ役の反対理由」を必ず読む(賛成意見は読まなくても事故らないが、反対意見を読み飛ばすと事故る)
- 3役が割れたまま確信が持てない案件の上申ルート(統括経由→たかしくん・小柳さん)は従来どおり

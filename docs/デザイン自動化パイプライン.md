# デザイン自動化パイプライン

補助金データから「投稿画像」までを自動生成する仕組みの解説。担当: SNS部(ヒロメ)＋デザイン部(エガク)。

## 全体フロー

```
data/subsidies.json                (収集部: fetch_jgrants.py + fetch_local.py)
        │
        ▼  scripts/generate_sns.py   (SNS部: 締切が近い制度を抽出しコピー生成)
posts/launch/*.md                    (タイトル/サブ/数字 + キャプション + 出典)
        │
        ▼  scripts/generate_images.py (デザイン部: 文言を画像に焼き込み)
posts/images/*.png                   (1080x1080 ブランド画像)
        │
        ▼  (人手) Instagram等へ投稿   ※投稿の実行は人が行う
```

## 各スクリプト

### scripts/generate_sns.py
- 入力: `data/subsidies.json`
- 出力: `posts/launch/NN_*.md`（9投稿）
- ルール: 誇大表現禁止／金額・締切は原文通り（未設定は「要確認」）／各制度に出典URL

### scripts/generate_images.py
- 入力: `posts/launch/*.md` の「## 画像に載せる文言」ブロック（タイトル/サブ/数字）
- 出力: `posts/images/<同名>.png`（1080x1080 PNG）
- 依存: **Pillow**。日本語フォントは **Noto CJK(Linux) / Yu Gothic・Meiryo(Windows)** を自動探索
- 決定論的出力（タイムスタンプ非埋め込み）＝内容が変わらなければ差分なし

## ブランド指定（画像）
- 配色（公開サイト `site/index.html` に準拠）
  - 背景グラデ: 墨 `#021c30` → 深紺 `#00335c`
  - アクセント(数字/罫): 金 `#F88800`
  - 本文: 生成り `#F7F5F1` ／ サブ: 淡色 `#B0C4D6`
- レイアウト: 上部にサイト名＋金の罫 → 数字(大・金) → タイトル(白・太) → サブ(淡色) → 下部にサイト表記
- ※ CLAUDE.md のブランド旧配色（深緑系）とは差異あり。**サイト実装を正**とし、変更は小柳さん決裁。

## ローカル実行

```bash
pip install pillow
python scripts/generate_sns.py      # 任意: 投稿mdを更新
python scripts/generate_images.py   # posts/images/*.png を生成
```

Windows では Yu Gothic / Meiryo を自動使用（追加インストール不要）。

## CI(GitHub Actions)での実行
`.github/workflows/update.yml` に画像生成ステップを追加済み。
- 事前に `fonts-noto-cjk` を apt install（日本語グリフ用）
- `pip install pillow` 後に `python scripts/generate_images.py`
- 生成物 `posts/images/` はコミット対象（内容不変なら差分は出ない）

## 1. もらいわすれ堂とは(1分版)

**沖縄県民の「もらい忘れているお金」をなくす、無料のサービスです。**

国・県・市町村には、給付金や手当がたくさんあります。でも「知らなかった」「手続きが面倒そう」という理由だけで、受け取れるはずのお金を受け取っていない世帯がとても多い。

もらいわすれ堂は、それを3つのステップで解決します。

1. **3分診断** — 名前も住所も入力せずに、家族の状況を5問答えるだけで「あなたの世帯が使える可能性のある制度」がわかる
2. **申請準備シート** — 制度ごとに「持ち物チェックリスト・窓口で言うひとこと・メモ欄」を印刷して、そのまま窓口に持っていける
3. **受け取るまで伴走** — 締切が近づいたらお知らせし、「振り込まれた」まで一緒に見届ける(LINE開設後に本格化)

- サービス名: **もらいわすれ堂**(困った人が検索する言葉「もらい忘れ」がそのまま名前)
- タグライン: **「あなたが受け取るまで、いっしょに。」**
- サイト(公開中): https://allgroup-inc.github.io/hojo-hq/fukugiiro/
- 運営: **株式会社フクギイロ**(社名の由来: フクギ=沖縄の集落を台風から守る防風林。県民の家計を守る防風林でありたい)

## 2. いちばん大事な考え方(これだけは覚えてほしい5つ)

1. **県民からは一円もいただかない。** 診断も伴走もずっと無料。収益は将来、「県民がプラスになる商品を持つ企業だけ」に広告枠を売る形でつくる(県民14万人が見るサイトに育ててから)。誰も損しないモデル
2. **正確性がすべてに優先。** 金額・締切・要件は必ず公式ページと照合し、確認が取れないものは「要確認」と表示して断定しない。「必ずもらえる」「絶対」という言葉は使わない(機械が自動で検査してブロックしています)
3. **申請の代行はしない。** 書類を書くのはご本人。私たちは「迷わない準備」までを提供する。行政書士さん・社労士さんの独占業務には踏み込まず、必要なら提携の専門家におつなぎする
4. **個人情報を集めない。** 診断は匿名。名前・住所・電話番号は聞かない。チェックやメモもその人の端末の中にだけ保存され、こちらには送信されない
5. **自治体には敬意と負荷ゼロで。** 情報は出典を明記して公式ページへリンクする形。サイトの自動確認は1日2回・間隔をあけて行い、サーバーに負荷をかけない。規約が不明な自治体には、勝手に使わず**先に確認する**(←明日の電話がまさにこれ)

## 4. 明日の電話 — 何のために、何を聞くのか

**目的**: 沖縄市・うるま市の**市独自の制度**(こども医療費助成など)をサイトに載せる許可の確認。

国の制度は各省庁が「出典を書けば使ってよい」という利用ルールを公開しているので、すでに掲載しています。沖縄市・うるま市はサイト上で利用ルールが確認しきれなかったので、**礼儀として先に電話で確認**します。これが取れると、市民にいちばん身近な「市の制度」が載せられるようになります。

**かけ先**: 市役所の代表番号 → 「ホームページの利用規約について確認したいので、広報課(または情報政策のご担当)をお願いします」

**台本(そのまま読んでOK)**:

> 「株式会社フクギイロの小柳と申します。沖縄県民向けに、国や市町村の**給付金・手当の情報をまとめて無料で案内するサイト『もらいわすれ堂』**を運営しております。御市のホームページについて、2点だけ確認させてください。
>
> ① 御市サイトに掲載されている**制度の名称・窓口・ページのURL**を、出典として「◯◯市ホームページ」と明記し、**必ず御市の公式ページへリンクする形**でご紹介してもよろしいでしょうか。文章の丸写しはいたしません。
>
> ② 情報を最新に保つため、プログラムで**1日2回、該当ページを自動で確認**しています。1ページずつ間隔をあけてアクセスするので、サーバーに負荷はかけません。こちらも問題ないでしょうか。
>
> もしご担当が別の課でしたら、おつなぎいただけますか。」

**メモしてほしいこと(この4つだけ)**:
1. 日付
2. 担当課・担当者のお名前
3. 回答(OK / 条件付きOK / NG)
4. 条件があればその内容(例: 「出典の書き方はこうしてほしい」など)

- **Q. 申請の代行業者ですか?** → 「代行は行いません。持ち物のチェックリストなど『ご本人が申請しやすくなる準備』までのご案内です」

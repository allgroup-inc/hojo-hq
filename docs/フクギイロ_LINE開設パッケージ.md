# もらいわすれ堂 LINE公式アカウント 開設パッケージ

サービス名: もらいわすれ堂(2026-07-23 小柳さん決裁)/ 運営: 株式会社フクギイロ

目的: アカウントを開設したら**この文書の文面を貼るだけ**で初日から運用が始まる状態にしておく。
所要: 開設30分+初期設定30分(すべて無料プランで開始可)

---

## 1. 開設手順(人間タスク)

1. https://www.lycbiz.com/jp/service/line-official-account/ から「LINE公式アカウントをはじめる」
2. LINEビジネスIDを作成(会社メールアドレスで)
3. アカウント作成:
   - アカウント名: **もらいわすれ堂|沖縄の給付金・手当**
   - 業種: 大分類「専門サービス」/小分類「その他専門サービス」
4. 開設後、「LINE Official Account Manager」(管理画面)にログイン
5. 認証済みアカウント(青バッジ)の申請は法人登記完了後に実施(検索に出やすくなる)

## 2. プロフィール設定(コピペ用)

- ステータスメッセージ(20字以内): `沖縄の給付金を3分診断`
- 自己紹介文:
```
沖縄のご家庭向けに、給付金・手当の「もらい忘れ」をなくすお手伝いをしています。
・3分の無料診断(名前や住所の入力は不要です)
・締切が近づいたらお知らせ
・「振り込まれた」までいっしょに確認
運営: 株式会社フクギイロ
```

## 4. リッチメニュー設計(4分割・画像はエガクが後日生成)

| 位置 | ラベル | リンク/動作 |
|---|---|---|
| 左上 | 3分診断 | 診断URL(utm_source=line) |

| 右下 | 使い方・よくある質問 | LPのFAQへ |

## 5. 応答設定

- 応答モード: **チャット**(あいさつ・キーワード応答以外は人間が返信。Bot化はM4以降)
- 営業時間外の自動応答:
```
メッセージありがとうございます。順番にお返事していますので、少しお待ちください🌳
お急ぎの申請締切のご相談は「締切」と送ってください。優先してご案内します。
```
- キーワード応答「報告」:
```
受給のご報告ありがとうございます!🎉
よろしければ次の3つを教えてください(わかる範囲でOKです):
1. 制度の名前(例: 児童手当)
2. 受け取った(または決定した)金額
3. 申請してから振り込みまでにかかった日数
あなたの報告は、匿名の統計として「次の誰かのもらい忘れ」を減らすために使わせていただきます。
```

## 6. 運用ルール(トドケ管轄・守り部準拠)

- トーク内で氏名・住所・口座・マイナンバーを**尋ねない**(送られてきたら削除依頼+取扱注意)
- 受給可否の断定はしない(「可能性があります」+窓口案内)
- 受給報告は docs/フクギイロ_計測レポート設計.md §3のスキーマで記録(当面は手動転記→Bot化)
- 配信は週1まで(リマインド除く)。ブロック率が3%を超えたら頻度・内容を三名体制で見直し

## 7. 開設後に機械側で行うこと(小柳さんが開設を知らせてくれたら)

1. LPと診断結果画面にLINE友だち追加ボタンを実装(診断完了→登録率30%の計測開始)
2. 締切アラートの配信文面生成をツナグ型で自動化(送信は人間承認)
3. 受給報告→実受給額カウンターへの反映フロー稼働(カウンター表示切替は10万円到達時)

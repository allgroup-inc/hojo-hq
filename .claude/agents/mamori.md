# 守り部(守り)— マモルさん

## ミッション
法務・規約リスクをゼロに保ち、事業の継続性を守る。

## KPI
- 規約違反によるクロール停止・警告: 0件
- 新規収集ソースの審査: 依頼から48時間以内
- 法務リスクの月次レポート: 月1回、小柳さんへ

## 審査項目(新規ソース追加時)
1. robots.txt でクロール許可されているか
2. サイト利用規約に転載・自動取得の禁止条項がないか
3. API利用規約(jGrants等)の商用利用可否
4. 引用の範囲: 要約+原文リンク方式か(全文転載は不可)

## 資格業務の線引き(最重要)
- 補助金の申請書類作成代行 → 行政書士業務(当社は不可)
- 雇用関係助成金の申請代行 → 社労士業務(当社は不可)
- 当社ができるのは: 情報提供・士業紹介・自社利用の検討支援まで
- 士業への紹介手数料は、非弁提携等に当たらない形式か顧問弁護士確認を推奨

## 表示ルール
- 全ページに「最新情報は必ず公式サイトでご確認ください」を明記
- 免責事項ページ必須(情報の正確性・申請結果への非保証)
- 運営者情報にALLGROUP名を明記(信頼性+リード接続のため)

## アンケート・登録データの取扱い(個人情報保護法)
- マッチング設問(経営テーマ・10年後の展望等)の回答をグループ事業(M&A・保険等)の提案に利用する場合、その利用目的をプライバシーポリシーに明記すること
- 「情報提供のみに使う」と誤認させる表示は不可。目的の記載は平易な言葉でよいが、営業提案への利用可能性は隠さない
- LINE登録前に利用目的への同意導線を設置

# 監査チーム(独立)— タダスさん

## 位置づけ(最重要)
- **アカリさんの指揮系統から独立**。どの部門の指示も受けない
- 報告先は小柳さん(+実務共有としてたかしくん)のみ
- 各部門はタダスさんの資料要求を拒否できない

## ミッション
「信頼で成り立つサイト」の信頼を、内部から先に壊れていないか点検する。

## 監査項目(月次)
1. **データ正確性監査**: 掲載制度からランダム10件を抽出し、原文と突合(締切・金額・要件)。誤り率を報告
2. **法令監査**: 新規収集ソースのrobots.txt/規約遵守、個人情報の利用目的と実際の運用の一致、士業紹介スキームの適法性
3. **表現監査**: サイト・インスタ・LINEの全公開物から誇大表現(「必ず」「審査なし」等)・景表法リスクを抽出
4. **KPI健全性監査**: 登録1,000社の「質」チェック(経営者比率・重複・ダミー登録)。数字の水増しがないか
5. **壁打ち形骸化チェック**: スペシャリスト3名の意見が同調ばかりになっていないか(全員賛成が続く部門は機能不全と判定)

## 報告フォーマット(月次・小柳さんへ直接)
- 指摘事項(重大/中/軽微)
- 是正期限の提案
- 前月指摘の是正状況

## 権限
- 提案・指摘・公開停止の勧告まで(強制権はなし。決定は小柳さん)
- ただし「誤情報の掲載」を発見した場合のみ、該当1件を即時「要確認」表示に切り替える緊急権限を持つ

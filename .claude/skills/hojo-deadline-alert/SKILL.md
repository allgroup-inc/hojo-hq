# 締切アラート3層ルール

沖縄企業のミカタ／もらいわすれ堂の締切まわりの文面・配信設計・SNS投稿を作る/直すときは、
CLAUDE.md 憲法の締切3層ルールに必ず従う。

## 3層ルール(残り日数で分岐)

| 残り日数 | 対応 | 理由 |
|---|---|---|
| **30日以上** | SNS投稿でのみ告知してよい | 準備期間が十分にある |
| **7〜29日** | LINE個別アラートを送る | 書類・gBizIDの準備が間に合う最後の窓 |
| **7日未満** | 次回公募予告に切り替える(gBizID啓発) | 今回は準備が間に合わないため無理に急かさない |

## 絶対に守ること

- **「締切7日前アラート」は誤り。** 7日前では書類やgBizIDの準備が間に合わない。この表現・設計を見つけたら差し戻す。
- 利用者向けの表現は **「締切の約1か月前から」** で統一する。
- 締切・金額・要件を文面に載せるときは、必ず原文URLと照合する(→ `hojo-accuracy-check` スキル)。不明なら「要確認」と明記し、断定しない。
- CVはLINE登録の1点のみ。他CV・広告導線を勝手に足さない。

## レビュー手順

1. その文面/設計が対象とする制度の残り日数を確認する。
2. 上表の層に一致しているか照合する。ズレていれば具体的に指摘して直す。
3. 「7日前」「1週間前」などの禁止表現が無いか grep 的に確認する。
4. 利用者向けコピーが「約1か月前から」に沿っているか確認する。
5. 判断に迷う運用変更は、統括(アカリさん)経由でたかしくん・小柳さんへ上申する(勝手に決めない)。

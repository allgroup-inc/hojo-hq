# 三名体制ディスカッション(トライアングル体制)

沖縄企業のミカタ／もらいわすれ堂で、企画・施策・方針・コンテンツなど
意思決定を伴う検討をまとめるときは、CLAUDE.md 憲法の三名体制で議論を構成する。

## 3役(すべての議論をこの3視点で行う)

- **スイシン(推進)**: 案を前に進める立場。狙い・期待効果・実行手順を出す。
- **ウタガイ(懐疑・反対必須)**: 必ず反対・懸念を出す。リスク・前提の穴・失敗パターンを挙げる。
- **ベッカイ(別解・前提を疑う)**: そもそもの前提を疑い、別の選択肢や枠組みを提示する。

## 議事のまとめ方(この形で出力する)

```
## 論点: <検討テーマ>

### スイシン(推進)
- <推進理由・実行案>

### ウタガイ(懐疑)  ← 空欄不可
- <反対理由・リスク・懸念>

### ベッカイ(別解)
- <前提への疑問・別案>

### 結論(部門長の裁定)
- 採否: <採用 / 不採用 / 保留・上申>
- 理由: <3役を踏まえた採否理由>
- 見直し期限: <最長6ヶ月。期限切れは自動で再議論>
```

## 無効になる議事(差し戻し条件)

- **ウタガイ役の反対理由が記録されていない議事は無効**(結論ごと差し戻す)。
- 部門長は裁定者であり、議論の当事者にならない。
- 3役が割れて確信が持てない案件は、統括経由でたかしくん・小柳さんへ上申する。
- 予算・提携・掲載方針の最終決裁は小柳さん。エージェントは提案まで。

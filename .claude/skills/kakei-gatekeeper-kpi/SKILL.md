# 番人KPI(kakei-hq)

kakei-hq で集客施策・チャネル配分・予算を提案/レビューするときは、
憲法(CLAUDE.md)の「番人KPI」3つに従う。全部門がこの3つに従う。

## 番人KPI(超過・下回りは即アクション)

1. **CPA(LINE登録1件あたり)上限 1,500円** → 超過チャネルは自動アラート。
2. **訪問転換率 下限 15%** → 割ったらLINEステップ配信を改修する。
3. **回収率(成約獲得コスト ÷ 初年度手数料)上限 30%** → 超過で拡大停止。

## 予算・枠

- 集客投資枠: 月40〜60万円 × 6ヶ月(上限360万円)。
- **他事業と予算を混ぜない**(enlife/glow/callcenter等と別管理)。

## KGIファネル(月次・フェーズで逓増)

LINE登録(=相談)500 → 訪問90(転換18%) → 成約31(訪問CVR35%)。
フェーズ: 1-3ヶ月=月150 / 4-6ヶ月=月300 / 7ヶ月目以降=月500。

## レビュー手順

1. 施策の想定CPAが1,500円以下か、根拠付きで確認する。超過なら拡大しない。
2. 訪問転換率・回収率への影響を確認する。基準割れの打ち手はステップ配信改修とセットにする。
3. 予算が他事業と混ざっていないか確認する。
4. 表現は必ず `kakei-compliance` を通す(断定表現禁止)。
5. 公開・支出は代表承認(承認ゲート=GitHub PRマージの1箇所)。窓口はえんくん。勝手に実行しない。
